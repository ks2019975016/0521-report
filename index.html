1. Todoist 앱의 편리성에 대한 소개

Todoist란?

todoist란 웹, store 앱, 모바일 앱 등 다양한 플랫폼에서 사용가능한 to-do 앱입니다.

3월 말 쯤에 있었던 졸업생 특강에서 추천해주셨던 앱이기도 합니다.

(개인적으로 느낀) 이 앱의 편리한 점들

1. 편리한 추가 - 추가시 내용과 함께 아래의 내용들을 쓰면 됩니다

- 느낌표 두 개와 숫자로 중요도 표시 (!!1 ~ !!4)

- 'OO월 OO일, OO마다, 다음주 O요일, OOO까지' 같은 글자를 쓰기만 해도 자동으로 일정이 설정됨

- '#(프로젝트이름)'으로 프로젝트에 편하게 추가

2. 프로젝트 기능

각 할 일들을 프로젝트에 추가하여 프로젝트별로 묶어볼 수 있다.

2. "모두의 알고리즘 with 파이썬" 책 소개

제가 이 책을 접한 건 고등학생 때였는데요,

그때 저는 그저 호기심만 조금 있었을 뿐 프로그래밍을 구체적으로 어떻게 되는건지, 파이썬이 무엇인지에 대해서 몰랐었는데도 불구하고 설명이 너무 자세하고 친절해서 어떤 식으로 프로그램이 만들어지고 돌아가는건지에 대해서 어렵지 않게 흐름을 파악할 수 있게 해준다는 점이 너무 좋았습니다.

가벼운 느낌으로 여러가지 내용들에 대해서 흥미와 호기심으로 읽어보고 싶다 하시는 분들께 추천드립니다.

목차

    들어가는 글

    [첫째 마당] 알고리즘 기초

  1. 1부터 n까지의 합 구하기
  2. 최댓값 찾기
  3. 동명이인 찾기 1

    4. [둘째 마당] 재귀 호출

  4. 팩토리얼 구하기
  5. 최대공약수 구하기
  6. 하노이의 탑 옮기기

    7. [셋째 마당] 탐색과 정렬

  7. 순차 탐색
  8. 선택 정렬
  9. 삽입 정렬
  10. 병합 정렬
  11. 퀵 정렬
  12. 이분 탐색



    [넷째 마당] 자료 구조

  1. 회문 찾기 (큐와 스택)
  2. 동명이인 찾기 2 (딕셔너리)
  3. 친구의 친구 찾기 (그래프)

    4. [다섯째 마당] 응용문제

  4. 미로 찾기 알고리즘
  5. 가짜 동전 찾기 알고리즘
  6. 최대 수익 알고리즘
